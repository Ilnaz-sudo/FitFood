FitFood — это мобильное приложение для автоматического подсчета калорий и нутриентов (белков, жиров, углеводов) с помощью компьютерного зрения. Просто сфотографируйте свою еду, и приложение сделает всё остальное!

Планируемый функционал
Распознавание еды через камеру: Пользователь может навести камеру на тарелку с едой и получить анализ ее состава.

Анализ из галереи: Загрузка и анализ фотографий еды из библиотеки устройства.

Подсчет нутриентов:
Автоматический расчет и отображение калорийности, содержания белков, жиров и углеводов (БЖУ) для распознанной еды.

История приемов пищи: Сохранение результатов для отслеживания питания в течение дня.

Учет личных целей: Возможность установить дневную норму калорий и следить за ее выполнением.

Техническое видение: Подсчет нутриентов с помощью Computer Vision
Планируется реализовать двумя основными способами:

1. Облачный API
Использование готовых решений: Интеграция с мощными API, такими как Google Cloud Vision AI или Amazon Rekognition, для классификации изображений еды.

Собственный ML-бэкенд: Развертывание собственной предобученной модели (например, на основе TensorFlow или PyTorch) на сервере. Приложение будет отправлять изображение на сервер и получать JSON-ответ с распознанными продуктами и их параметрами.

2.On-Device ML (Конфиденциальность, работа офлайн)
Интеграция TensorFlow Lite: Использование легковесных моделей машинного обучения, которые работают прямо на устройстве пользователя. Это обеспечит максимальную конфиденциальность данных (изображения никуда не отправляются) и позволит работать без подключения к интернету.
